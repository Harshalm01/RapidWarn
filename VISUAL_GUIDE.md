## 🎮 Build Menu Options (What Each Does)

```
┌─────────────────────────────────────────────────────────┐
│  📱 RapidWarn - Build Menu                              │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  1. Build Debug APK                                     │
│     → Just creates APK file                             │
│     → Use: Rarely (when you just need the file)        │
│                                                          │
│  2. Build + Install ⭐ MOST USED                        │
│     → Builds + Puts on phone                            │
│     → Use: Every day, all day!                          │
│                                                          │
│  3. Clean + Build + Install                             │
│     → Fresh start from scratch                          │
│     → Use: When build fails or weird errors            │
│                                                          │
│  4. Run with Hot Reload                                 │
│     → Live coding (change & see instantly)              │
│     → Use: When making UI changes                       │
│                                                          │
│  5. Check Device Connection                             │
│     → Is phone connected?                               │
│     → Use: When phone not detected                      │
│                                                          │
│  6. View App Logs                                       │
│     → See what's happening inside app                   │
│     → Use: When app crashes or debugging                │
│                                                          │
│  7. Setup Google Sign-In                                │
│     → Runs the Google setup wizard                      │
│     → Use: Once, or when Google login fails             │
│                                                          │
└─────────────────────────────────────────────────────────┘
```
